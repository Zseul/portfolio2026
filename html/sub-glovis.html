현대글로비스 웹사이트는 React와 Next.js 기반 환경에서 진행된 프로젝트로, 컴포넌트 단위 구조와 데이터 기반 UI 렌더링 방식 등 프론트엔드 개발 프로세스를 경험할 수 있었던 작업입니다. API를 통해 전달받은 데이터를 화면에 바인딩하고, 상태 값에 따라 UI가 동적으로 변화하도록 구현하며 데이터 흐름과 컴포넌트 간 구조를 이해할 수 있었습니다.

또한 페이지 단위 렌더링 구조와 재사용 가능한 컴포넌트 설계를 경험하였으며, 프론트엔드 개발자와의 협업을 통해 React 기반 프로젝트의 구조와 작업 방식을 익힐 수 있었습니다.
